◆ squareAttacked()

uint64_t squareAttacked ( const Gamestate * gamestate,
const Position * position,
const bool checkForWhite )

returns bitboards containing all checking pieces + all block possibilities + all king moves for the current gamestate color. If checkForWhite is true the return value will be the position of black pieces attacking the square. If this is 0 while king is in check => checkmate.

also saves the bitboards into config of the gamestate

Parameters
gamestatethe gamestate to check
kingPositionthe position of the king that needs to be checked
checkForWhitetrue for white, false for black
Returns
: bitboard with all possible king moves, piece captures or block opportunities
